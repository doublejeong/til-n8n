
### Context
프로그래밍 학습의 가장 기본적인 시작은 "Hello World" 프로그램을 작성하는 것입니다. 이 문서는 파이썬(Python) 언어에서 "Hello World"를 출력하는 방법을 설명하고, 기본적인 파이썬 코드 실행 과정을 이해하는 데 도움을 줍니다.

### Core
파이썬에서 "Hello World!" 문자열을 콘솔에 출력하는 가장 간단한 방법은 `print()` 함수를 사용하는 것입니다.

```python
print("Hello World!")
```

### Insight
위 코드를 실행하면 `Hello World!`라는 문자열이 화면에 출력됩니다. `print()` 함수는 파이썬의 내장 함수(built-in function)로, 괄호 `()` 안에 전달된 내용을 표준 출력(standard output)으로 보내는 역할을 합니다. 이 간단한 코드는 파이썬 스크립트 작성 및 실행의 기본 원리를 보여주며, 모든 프로그래머에게 필수적인 첫걸음입니다.
