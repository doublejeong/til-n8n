### Context
파이썬에서 새로운 함수를 작성하여 테스트할 계획입니다. 이 함수는 리스트에서 모든 짝수를 찾아 반환해야 합니다.

### Core
```python
def find_even_numbers(numbers):
    even_numbers = []
    for number in numbers:
        if number % 2 == 0:
            even_numbers.append(number)
    return even_numbers

# 테스트할 숫자 리스트
test_numbers = [1, 4, 5, 8, 9]

# 함수 호출 및 결과 확인
result = find_even_numbers(test_numbers)
print(result)  # 예상 출력: [4, 8]
```

### Insight
위의 코드는 입력된 숫자 리스트에서 짝수만을 찾아 반환하는 기능을 구현하였습니다. 테스트를 통해 예상한 결과와 일치하였으므로, 이 함수는 올바르게 작동하고 있다고 확인할 수 있습니다.
