### Context
CSS에서 이미지 위에 다른 이미지를 레이어링하려면, absolute positioning을 사용할 수 있습니다. 이 기술은 상위 컨테이너의 relative position과 하위 요소의 absolute position을 조합하여 가능하며, 이렇게 하면 하위 요소가 상위 요소의 위치에 정확하게 맞추어 배치됩니다.

### Core
상위 요소를 레일레이션으로 설정하고 하위 요소를 로 설정하여 이미지 위에 다른 이미지를 배치할 수 있습니다. 예를 들어:

```html
<div class="container">
  <img src="background-image.jpg" alt="" class="bg-img">
  <div class="overlay">
    <img src="overlay-image.png" alt="" class="overlay-img">
  </div>
</div>
```

스러한 구조에서, `.container`는 상위 요소이며 relative position을 가지고 있어야 하며, `.overlay`는 absolute position을 가지도록 하여 배치 위치를 정확하게 제어할 수 있습니다. `.overlay-img`는 `.overlay` 안에 위치하므로, `.overlay`의 absolute positioning이 .`overlay-img`의 상대 위치를 결정합니다.

```css
.container {
  position: relative;
  width: 300px;
  height: 200px;
}

.overlay {
  position: absolute;
  top: 10px;
  left: 10px;
}

.overlay-img {
  display: block;
}
```

위의 예에서는 `.overlay` 요소가 `.container` 안에서 10px 왼쪽, 10px 위에 위치하게 됩니다.

### Insight
위 코드는 상위 이미지(`.bg-img`)와 하위 이미지(`.overlay-img`)를 겹쳐 배치하는 방법을 보여줍니다. 이를 통해 다양한 효과를 만들 수 있습니다. **출처:** [CSS Positioning Guide](https://developer.mozilla.org/en-US/docs/Web/CSS/position)
