### Context
제미나이(Jest)는 JavaScript 프론트엔드 개발에서 널리 사용되는 테스팅 라이브러리입니다. 클라우드 함수를 활용하여 제미나이 테스트 코드를 작성하고 실행하는 방법을 알아보겠습니다.

### Core
```javascript
// jest.test.js
const add = (a, b) => a + b;

describe('add function', () => {
  it('should return the correct sum of two numbers', () => {
    expect(add(1, 2)).toBe(3);
  });
});

export default add;
```

위 코드는 `add` 함수의 정확성을 테스트하는 간단한 제미나이 테스트입니다. 이를 클라우드 함수로 실행시키면, 테스트 결과를 실시간으로 확인할 수 있습니다.

### Insight
**출처:** [Jest Documentation](https://jestjs.io/docs/getting-started)
